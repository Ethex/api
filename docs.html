Ethex API Documentation

Smart Contract

Ethex stores all of it's market data (orders and trades) in the Ethereum blockchain. Our smart contract is a low level API that allows you to do any important market related function.

Contract source at github

Market API

The API here is provided for convenience. This functionality is provided by reading data off the Ethereum blockchain. You can run this on your own machine against your own Ethereum node or against a public provider like Etherscan.
To run api, download from our github. When running api and you enter "0" ("npm start 0") the min eth for open orders will be zero instead of the standard 0.001.

API source at github

$ git clone https://github.com/Ethex/api
$ cd api
$ npm install
$ node server.js

Market Data

Once the server is running you can query it for market data. Its provided in JSON format.

/ticker24

returns a dictionary of ETH_Symbol to 24 hours of market data.
  • last: the price of the last trade.
  • lowestAsk: the lowest price offered to buy a token for.
  • highestBid: the highest price offered to sell a token for.
  • high24hr: The highest price paid in the last 24 hours in Ether.
  • low24hr: The lowest price paid in the last 24 hours in Ether.

/openOrders

returns all the market orders
  • hash: An order ID that can be used to cancel.
  • type: Buy/Sell
  • tokenAmount: The size of the order in tokens.
  • ethAmount: The total amount of ETH.
  • priceEth: The price in ETH for a token.
  • symbol: The symbol of the token.
  • tokenAddress: The Ethereum address of the token.
